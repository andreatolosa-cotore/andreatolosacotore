# andreatolosacotore
Andrea Tolosa Cotoré es artista, docente y doctoranda en Educación (UNIZAR). Vincula creación y pedagogía desde los nuevos materialismos. Con base técnica y artística en videoarte y escultura, impulsa el diálogo crítico y social. Como fundadora de Materia Sensible, transforma experiencia y emoción en conocimiento.
